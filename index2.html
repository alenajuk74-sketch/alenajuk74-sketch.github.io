Тестирование API

Имею практический опыт тестирования REST API с использованием Postman. В рамках учебных проектов работал с различными типами запросов, проверял корректность ответов, статус-коды, структуру данных и обработку ошибок.

Основные навыки в тестировании API:

  • Создание и отправка HTTP-запросов (GET, POST, PUT, DELETE, PATCH)
  • Работа с параметрами запросов (query params, path variables, headers, body)
  • Настройка заголовков (headers) и тела запросов (JSON, XML, form-data)
  • Проверка статус-кодов ответов (200, 201, 400, 401, 403, 404, 500 и др.)
  • Валидация структуры ответов (JSON Schema), проверка типов данных и значений
  • Работа с аутентификацией (Bearer token, Basic auth, OAuth 2.0)
  • Написание простых тестов на JavaScript в Postman
  • Организация запросов в коллекции, использование переменных и окружений
  • Тестирование граничных значений и негативных сценариев
